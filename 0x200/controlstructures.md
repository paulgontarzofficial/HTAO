# Control Structures

**If-Then-Else**

```
If (condition) then
{
  Set of instructions to execute if the condition is met;
}
Else
{
  Set of instructions to execute if the condition is not met;
}
```

**While/Until Loop**

```
While (you are hungry)
{
  Find some food;
  Eat the food;
}
```

**For Loops**

```
For (i=0; i<5; i++)
  Drive straight for 1 mile;
```
